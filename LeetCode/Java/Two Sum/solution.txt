Use HashMap to store the difference of target and element with element index. Then looping the array 
to find if there is a difference the same as any element in this array. 

class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<Integer, Integer>();
        int[] answer = new int[2];
        for(int i = 0; i < nums.length; i++) {
            if(map.containsKey(nums[i])) {
                answer[0] = map.get(nums[i]);
                answer[1] = i;
            }
            else {
                map.put(target - nums[i], i);
            }
        }
        return answer;
    }
}
